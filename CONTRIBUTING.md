# Contributions

I don't need any contributions to this website, but if you have ideas, I'd love to hear your feedback in the form of a [github issue](https://github.com/alexanderdombroski/alexanderdombroski.github.io/issues).

## Environment Variables

Some pages use a github token to improve the octokit github API rate limit.

Add a `.env`.

```bash
GITHUB_TOKEN=github_pat_<token>
```
